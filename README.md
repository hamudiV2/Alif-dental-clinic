# Alif Dental Clinic Website

**Live Site:** [alifdentalclinic.com](https://alifdentalclinic.com)

## Overview

The **Alif Dental Clinic** website is a modern, responsive site designed for a dental clinic client. It showcases services, team members, appointment booking, and contact information to help attract and engage patients.

## Features


- **Responsive Design:** Optimized for mobile, tablet, and desktop devices
- **Service Pages:** Details on offered dental services
- **Team Section:** Profiles of dentists and staff
- **Appointment Booking:** Easy-to-use form for scheduling visits
- **Contact Information:** Maps, emails, phone, and address
- **SEO Optimization:** Structured content for better search results
- **Fast Performance:** Lightweight, quick loading pages


## TechStack Used

- **HTML5/CSS3**
- **JavaScript**
- **Bootstrap** 
- **React.js**
